# Task 3: Interactive DUAL AXIS Power BI Dashboard

## Objective

Develop an interactive Power BI dashboard to visualize key metrics from the Google Play Store dataset.

## Tools & Technologies

* Power BI
* DAX
* Power Query

## Steps Performed

* Imported the cleaned dataset into Power BI.
* Performed data transformation using Power Query.
* Created DAX measures for key metrics.
* Designed interactive charts and KPIs.
* Added slicers and filters for better user interaction.

## Output

An interactive dashboard providing insights into app categories, installs, ratings, reviews, and pricing.
