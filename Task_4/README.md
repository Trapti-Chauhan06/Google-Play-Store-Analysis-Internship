# Task 4: Time Series Analysis Dashboard

## Objective

Create a time-based dashboard to analyze app installs over different time periods.

## Tools & Technologies

* Power BI
* DAX
* Date Table

## Steps Performed

* Created a Date Table.
* Developed DAX measures for total installs.
* Built a time-series line chart.
* Implemented date-based filtering.
* Enabled interactive analysis using slicers.

## Output

A dynamic dashboard showing trends in app installs across different time periods.
