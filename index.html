Google-Play-Store-Data-Analysis-Internship

End-to-End Data Analytics Project using Python, Plotly & Power BI

Project Overview

This project demonstrates the complete data analysis workflow on the Google Play Store dataset. It includes data cleaning, exploratory data analysis, interactive visualizations, and Power BI dashboard development to extract meaningful business insights.

Tools & Technologies

  • Python
  • Jupyter Notebook
  • Pandas
  • NumPy
  • Plotly
  • Power BI
  • Power Query
  • DAX
  • Microsoft Excel

Internship Tasks

Task_1

Data Cleaning & Preprocessing using Python and Pandas.

Task_2

Exploratory Data Analysis with interactive Plotly visualizations.

Task_3

Interactive Power BI Dashboard with KPIs, charts and slicers.

Task_4

Time Series Analysis Dashboard using Date Table and DAX.

Task_5

Bubble Chart Analysis to visualize installs, ratings and reviews.

Task_6

Stacked Area Chart for cumulative installs and growth trends.

Project Deliverables

  • Jupyter Notebook Files
  • Power BI Dashboards (.pbix)
  • Dashboard Screenshots
  • Task-wise Documentation
