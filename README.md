# Google-Play-Store-Analysis-Internship
End-to-End Google Play Store Data Analysis using Python, Plotly, and Power BI.
## Project Overview

This repository contains my completed internship tasks based on the Google Play Store dataset. The project covers the complete data analysis workflow, including data cleaning, exploratory data analysis (EDA), interactive visualizations, and Power BI dashboard development. The objective was to transform raw data into meaningful insights using Python and Power BI.

## Objectives

* Clean and preprocess the Google Play Store dataset.
* Perform exploratory data analysis to identify trends and patterns.
* Create interactive visualizations using Plotly.
* Build Power BI dashboards to analyze app performance.
* Apply DAX measures and time-based analysis for business insights.

## Dataset

* Google Play Store Apps Dataset
* Google Play Store User Reviews Dataset

## Project Structure

### Task 1 – Data Cleaning and Preprocessing

* Cleaned the dataset by handling missing values, removing duplicates, correcting data types, and preparing the data for analysis.

### Task 2 – Exploratory Data Analysis (EDA)

* Performed exploratory analysis and created interactive Plotly visualizations to understand app categories, installs, ratings, reviews, and pricing.

### Task 3 – Interactive Power BI Dashboard

* Built an interactive dashboard with KPIs, charts, slicers, and filters to analyze app performance.

### Task 4 – Time Series Analysis

* Created a Date Table, DAX measures, and a time-series dashboard to visualize app install trends over time.

### Task 5 – Bubble Chart Analysis

* Designed an interactive Bubble Chart in Power BI to analyze the relationship between installs, ratings, and reviews across app categories.

### Task 6 – Stacked Area Chart Analysis

* Developed cumulative measures and a Stacked Area Chart to visualize category-wise app growth over time.
